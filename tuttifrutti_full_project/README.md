
Tutti-Frutti - Proyecto base
===========================

Módulos incluidos:
- tuttifrutti-common
- tuttifrutti-server
- tuttifrutti-client
- tuttifrutti-ui

Cómo ejecutar (necesitas Maven y JDK 21LTS):

Link Descarga JDK 21 LTS
**https://adoptium.net/es/temurin/releases?version=21&os=any&arch=any**

Link Descarga Maven
https://maven.apache.org/download.cgi
(DESCARGUEN EL QUE DICE Binary zip archive)

- Ejecutar servidor (desde la raíz del proyecto):
  ./run-server.sh

- Ejecutar cliente de demo:
  ./run-client.sh
